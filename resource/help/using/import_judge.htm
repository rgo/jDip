Importing Judge Files

jDip allows the import of files created by the online email Judge system. The allowed files include: game history files (single or multiple turns), game listing files, and turn result files.

For more information about online Judges, see www.diplom.org.

Only variants supported by jDip may be imported.

How Importing Works

If a game is already open, Judge import first checks to see if the game name is the same. If the game name is the same as that being imported, and the game phase is the same, the imported turns will be added to the current game. Games with the same name but different phase can be rewound to the correct phase for importing to resume.

If no game is open, or, the name of the currently open game is not the same as that being imported, jDip will create a new game from the imported data.

If a game listing is imported, a starting position with appropriate player information will be created.

If a game history is imported (single or muliple turns), jDip will create a game with all imported turns.
