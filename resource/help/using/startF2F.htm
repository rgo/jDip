Starting A New Face-to-Face Game

To start a new game, select New | Face-to-Face Game... from the File menu. The New Face-to-Face Game window will then appear, with several tabs to set the type of game and options for that game.

Note that the options for a Face-to-Face game are the same as those for a regular game. See Starting a New Game for more information on these options.

What are Face-to-Face games?

A face-2-face game (F2F) is a game where all players are together, in the same room; using jDip, it is analagous to 'hot-seat' game, where each player enters his or her orders at the computer, one player at a time.

When a game is created as an F2F game, orders are entered in a restricted fashion. After a period of diplomacy, during which players may interact with each other, the players take turns entering their orders in secret. Players cannot see the orders another player has entered until the turn has been resolved.

Entering orders in a Face-to-Face game

When a tab with the desired power is clicked, only orders for that power may be entered. After a player has finished entering orders, he or she must submit them using the Submit Orders button. A confirmation window will then be displayed. After orders have been submitted, they cannot be changed in any way.

When all powers have submitted orders, the turn will resolve. Only when the turn has resolved are all orders visible.

A player may view the results and maps of previous turns using the History menu or appropriate keyboard shortcuts. When reviewing prior turns, a player may select the All tab to see orders for all powers. Otherwise, the All tab is not available.

A power's tab will be grayed-out (not selectable) when orders for that power have been submitted. A tab may also be grayed-out if a power has no orders to submit during a Retreat or Adjustment phase.
